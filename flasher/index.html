Build for an unlisted board

Any original ESP32 board with Bluetooth Classic support should work. ESP32-S3, C3, and C6 cannot connect to TH-D75 class radios over Bluetooth SPP and are not suitable for the radio bridge.

1. Clone and install PlatformIO

git clone https://github.com/__REPO__
cd sideband/firmware/sideband-bridge
python3 -m pip install --user platformio esptool

2. Add your board target to firmware/sideband-bridge/platformio.ini. Copy the [env:esp32dev-ble] section and change the board value to match your hardware. Board identifiers are listed at registry.platformio.org.

3. Build and create a merged binary

pio run -e my-board-ble

# Adjust flash parameters to match your board if needed.
# Check: pio run -e my-board-ble -v  (look for --flash_mode / --flash_freq flags)
BOOT_APP=$(find ~/.platformio -name "boot_app0.bin" -path "*/partitions/*" | head -1)

esptool.py --chip esp32 merge_bin \
  --flash_mode qio --flash_freq 40m --flash_size 4MB \
  -o merged.bin \
  0x1000  .pio/build/my-board-ble/bootloader.bin \
  0x8000  .pio/build/my-board-ble/partitions.bin \
  0xe000  "$BOOT_APP" \
  0x10000 .pio/build/my-board-ble/firmware.bin

4. Flash using the Flash a custom binary section above — select your merged.bin file and click Flash. Or flash directly with PlatformIO without needing the merged binary: 

pio run -e my-board-ble --target upload
